# 我的博客
## 仓库简介
此仓库含有两个branch: master和hexo(default), 其中hexo存放博客的一些hexo文件(即我本地的Myblog/blog文件夹所有内容)如md文件等等，而master存放发布到网站的有md文件生成的静态文件，即md博客文件写好后通过命令`hexo g -d`发布网站时会将静态文件到master分支。
